⚠️ Note: This project is currently under development. Some planned features (AI chatbot, crop image analysis) are not yet implemented.

📱 Features Implemented : 

🌍 Multilingual Support
Allows users to interact with the app in different languages.
☁️ Weather Information
Displays weather data using external APIs.
🌾 Crop Information
Provides basic information about crops to help farmers make informed decisions.
🏛️ Government Schemes (Static)
Displays predefined government schemes related to agriculture.


🚧 Features Planned (Not Yet Implemented) : 

🤖 AI Chatbot for farmer assistance
📷 Crop Image Scanning for analysis using photos
📡 Dynamic government schemes fetched from live sources

These features are planned for future development and are not functional yet.


🛠 Tech Stack :

*Android Studio
*Kotlin
*Jetpack Compose
*Firebase


🎯 Purpose of the Project :

This project aims to -
*Solve real-world problems in the agriculture domain
*Practice modern Android development using Jetpack Compose
*Build scalable app architecture with Firebase
*Improve accessibility through multilingual support


📌 Notes :

*This project is part of learning and experimentation.
*Code and features will evolve over time.
*Contributions and suggestions are welcome.
